# Simulation of a Counting Process

We simulate a counting process over a time interval \(T = 1\), where events occur independently and uniformly at a constant average rate \(\lambda\). The interval is divided into \(n\) small subintervals, generating an event in each with probability \(\lambda/n\).

<script>
function simulate() {
  const lambda = parseFloat(document.getElementById('lambda').value);
  const n = parseInt(document.getElementById('n').value);
  const T = 1;
  const dt = T/n;
  
  // Initialize counts
  let cumulativeCounts = [];
  let count = 0;

  for (let i = 0; i < n; i++) {
    if (Math.random() < lambda/n) {
      count += 1;
    }
    cumulativeCounts.push(count);
  }

  const labels = Array.from({length: n}, (_, i) => ((i+1)*dt).toFixed(4));

  if (window.countingChart) {
    window.countingChart.data.labels = labels;
    window.countingChart.data.datasets[0].data = cumulativeCounts;
    window.countingChart.update();
  } else {
    const ctx = document.getElementById('countingChart').getContext('2d');
    window.countingChart = new Chart(ctx, {
      type: 'line',
      data: {
        labels: labels,
        datasets: [{
          label: 'Cumulative Count',
          data: cumulativeCounts,
          borderColor: 'rgba(75, 192, 192, 1)',
          backgroundColor: 'rgba(75, 192, 192, 0.2)',
          fill: true,
          tension: 0.1
        }]
      },
      options: {
        responsive: true,
        plugins: {
          legend: { display: true },
          title: { display: true, text: 'Counting Process Simulation' }
        },
        scales: {
          x: { title: { display: true, text: 'Time' } },
          y: { title: { display: true, text: 'Cumulative Count' }, beginAtZero: true }
        }
      }
    });
  }
}
simulate();
</script>

## Analysis

This simulation approximates a **Poisson process**:

- **Stochastic Process:** Counts of events over time, with independent increments.
- **Properties:**
  - Number of events in disjoint intervals is independent.
  - The probability of one event in a small interval \(dt\) is approximately \(\lambda dt\).
  - Inter-arrival times follow an exponential distribution with mean \(1/\lambda\).
- **Interpretation of \(\lambda\):** Average rate of events per unit time. Higher \(\lambda\) means events occur more frequently.  

This discrete approximation becomes increasingly accurate as \(n \to \infty\), converging to the theoretical Poisson process.
